Lumière & Co. — demo jewellery store

A stand-in storefront recreating the chatbot integration built for a real D2C jewellery client. Click the chat bubble in the bottom right and try asking about rings, shipping, or a gift for a birthday.

Note: this demo runs entirely client-side with a rule-based response engine — no API calls, no real inventory. The production version integrates the Gemini API server-side (Flask), live product data, and Shopify checkout redirect.
